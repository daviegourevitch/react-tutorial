# react-tutorial

Welcome to the React tutorial!

Getting your environment set up:

1. Install node

   Head to this URL and download the file then install
   https://nodejs.org/dist/v22.16.0/node-v22.16.0-x64.msi

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npm run start
   ```
